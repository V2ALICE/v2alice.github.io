Hardware

The ALICE hardware architecture embodies a meticulously crafted system that integrates diverse components harmoniously. At the heart of ALICE's hardware lies the Raspberry Pi 4 (RPI4), an adaptable and robust single-board computer. Serving as ALICE's central processing unit, the RPI4 undertakes intricate AI computations and executes an array of tasks with exceptional speed and precision.

Seamless communication between the Raspberry Pi and an Arduino microcontroller forms a cohesive framework for controlling the actuators and sensors employed by ALICE. This synergy empowers ALICE to effortlessly interact with the physical environment, seamlessly managing and controlling household devices like lights, doors, curtains, and fans.

To enhance visual perception, ALICE incorporates a webcam that establishes a connection with the Raspberry Pi. This webcam facilitates real-time object and character recognition, enabling ALICE to interact with the environment and provide intelligent responses based on visual cues. Additionally, the integration of a microphone with the Raspberry Pi allows for the capture and processing of user voice commands. This seamless integration empowers ALICE's voice assistant feature, guaranteeing precise and efficient voice recognition capabilities. For a captivating and immersive experience, ALICE integrates a sound amplification unit connected to the Raspberry Pi's auxiliary output. This integration ensures top-notch audio output quality, enabling users to engage with ALICE through clear and audible sound.
